# UI-UX-Experiment6

### Name: RENUKA R

### Register No: 212223240136
## **Aim**

To design an integrated UX model that merges study content with playful interaction, encouraging consistent learning, skill-building, and increased student engagement.


## **Algorithm**

### **1. Understand User Goals**

Identify what learners expect from educational tools and game-based platforms.

### **2. Craft a Unified Interface**

Develop an experience where lessons, tasks, and game challenges operate smoothly together.

### **3. Add Key Functional Modules**

Implement progress insights, competitive modes, reward systems, and adaptive learning tools.

### **4. Test & Improve**

Gather feedback through usability checks to refine flow, clarity, and user satisfaction.

### **5. Launch & Evaluate**

Deploy the prototype, collect student and parent feedback, and refine based on real usage patterns.

---

## **1. Solution Ideation**

**Method Used:** Rapid Brainstorming

**Theme:** **"LearnJoy" – A smart learning platform powered by interactive play.**

### **Eight Generated Ideas:**

1. Personalized home dashboard with subjects, badges, and quick-start cards.
2. Concept-based learning games for each chapter.
3. AI-driven learning coach suggesting activities based on weak areas.
4. Daily XP goals with reward boosters and certificate unlocks.
5. Quiz battles to encourage friendly competition.
6. Voice-enabled support for guidance during gameplay.
7. Parent monitoring panel for tracking performance and consistency.
8. AR-enabled tasks for better visualization of topics.

## **3. Scenario Development**

### **Chosen User Story:**

**As a learner, I want to study through fun tasks so I stay engaged.**

### **Scenario:**

Maya, a 12-year-old student, logs into the **EduSpark** app after school. Her dashboard displays her progress in Social Science and Mathematics. The system suggests a "Treasure Trail Quiz" to improve her history knowledge. Maya completes the game, earns 35 XP, and unlocks a new reward badge. She checks her weekly ranking and feels motivated to continue.

**Emotional Flow:** Interested → Immersed → Confident → Motivated

**Environment:** Bedroom desk with tablet.

**Tasks:** Open app → Review dashboard → Choose suggested activity → Play → Earn rewards → Check ranking
